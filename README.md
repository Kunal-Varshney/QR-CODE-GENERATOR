# 🔳 QR Code Generator  

A simple and stylish QR Code Generator built with **HTML, CSS, and JavaScript**.  
It allows users to generate QR codes from text or URLs and download them instantly.  

---

## 🚀 Features
- Generate QR codes from any text or URL  
- Download QR code as PNG image  
- Stylish and clean user interface  
- Fast and responsive design  
- Custom QR colors for better look  

---

## 🛠️ Tech Stack
- **HTML** → Structure of the app  
- **CSS** → Styling and layout  
- **JavaScript** → Functionality and logic  
- **QRCode.js** → QR code generation library  

---

## 🎯 How to Use
1. Open the app in your browser  
2. Enter text or a URL in the input field  
3. Click on **Generate QR**  
4. Click on **Download QR** to save the image  

---

## 🖼️ Preview
- Input box to enter text or URL  
- Two buttons: **Generate QR** and **Download QR**  
- QR code displayed inside a styled box  
- Smooth UI with gradient background
